# 💰 FINOVA – Smart Wealth Console

FINOVA is a modern expense tracking and personal finance dashboard built with HTML, CSS, and JavaScript. It provides real-time financial insights, budget management, transaction tracking, and interactive analytics through a premium user interface.

## 🚀 Live Demo

🔗 https://theaadityatiwari-dev.github.io/Finova/

## ✨ Features

* 📊 Interactive financial analytics and charts
* 💵 Income & expense tracking
* 🎯 Budget management with category-wise limits
* 📈 Financial health score and spending insights
* 🔍 Search, filter, and sort transactions
* 🌙 Light/Dark theme support
* 💱 Multi-currency support
* 📱 Responsive design for desktop and mobile
* 📤 Export ledger functionality

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js
* Lucide Icons

## 📂 Project Overview

FINOVA helps users monitor their financial activities, analyze spending habits, manage budgets, and gain actionable insights through a sleek and modern dashboard experience.
